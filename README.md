# Titanic Dataset - Exploratory Data Analysis (EDA)

This project contains an exploratory data analysis of the Titanic dataset using Python. The goal is to understand the patterns in the data, such as factors that influenced passenger survival.

## 📊 Key Objectives
- Understand the structure and content of the Titanic dataset
- Clean and preprocess the data
- Perform univariate and bivariate analysis
- Visualize insights using various plotting techniques
- Analyze survival rates based on features like age, gender, class, etc.

## 🛠 Tools & Libraries Used
- Python
- pandas
- numpy
- matplotlib
- seaborn
- Jupyter Notebook
